Features

  • One-click bookmark saving
  • Add tags directly from the extension
  • Public/private bookmarks
  • Automatically captures page title and URL
  • Captures selected text as description
  • Works offline (saves when you're back online)

Installation (Manual)

  1. Download the extension from GitHub (link above)
  2. Extract the ZIP file
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" (toggle in top right)
  5. Click "Load unpacked"
  6. Select the chrome-extension folder
  7. Done! The extension is now installed
💡 Works on: Chrome, Microsoft Edge, Brave, and any Chromium-based browser that supports Manifest V3.

How to use

  1. Navigate to any webpage you want to save
  2. Click the OnlyLinks icon in your browser toolbar
  3. A popup will show the page details
  4. Add tags, adjust title/description
  5. Click "Save bookmark"
  6. Done!
💡 Pro tip: Select text on the page before clicking the extension icon - it will be used as the description!

Keyboard Shortcut (Optional)

You can set up a keyboard shortcut for even faster saving:

  1. Go to chrome://extensions/shortcuts
  2. Find "OnlyLinks - Quick Save"
  3. Set your preferred shortcut (e.g., Ctrl+Shift+D)
